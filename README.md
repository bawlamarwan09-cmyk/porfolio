# Premium Full-Stack Developer Portfolio

Premium, multilingual (EN / FR / AR + RTL) portfolio built with Next.js 15, React 19, TypeScript, Tailwind CSS, Framer Motion and Lucide.

## Features

- Dark, luxurious UI inspired by Apple / Linear / Stripe
- Glassmorphism, bento grids, animated gradients & cursor glow
- Custom loader, scroll progress, sticky blurred navbar
- Trilingual content (English, Français, العربية) with full RTL
- Bilingual FR/AR CTAs for premium feel
- Featured projects with bento layout + dynamic filtering
- Dedicated case study pages per project (problem → result)
- Services, About + timeline, Tech stack, Testimonials, Blog, Contact
- SEO: metadata, OG, Twitter, JSON-LD Person schema, sitemap, robots
- Mobile-first responsive

## Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Build

```bash
npm run build
npm start
```

## Customizing

- Site info: `src/data/site.ts`
- Projects (bilingual content + case studies): `src/data/projects.ts`
- Services: `src/data/services.ts`
- Posts: `src/data/posts.ts`
- Testimonials: `src/data/testimonials.ts`
- Tech & timeline: `src/data/tech.ts`
- UI strings: `src/i18n/dictionaries.ts`
